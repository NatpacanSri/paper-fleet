# Product

## Register

product

## Users

กลุ่มเพื่อนที่ต้องการเล่นเกมวางแผนออนไลน์ 2–6 คนผ่าน desktop หรือ mobile โดยเข้าเล่นได้เร็วด้วยชื่อเล่นและรหัสห้อง

## Product Purpose

ดัดแปลงเกมเรือรบบนกระดาษให้เป็นเกมออนไลน์ที่ยังรักษาการวางแผนลับ การจำข้อมูล การอ่านใจเพื่อน และจังหวะเปิดผลพร้อมกัน ความสำเร็จคือผู้เล่นเข้าใจกติกาจาก UI เอง จัดสนามได้โดยไม่ติดขัด และเล่นหลายรอบได้โดยไม่เสียจังหวะ

## Brand Personality

สนุก ทำมือ เป็นกันเอง เหมือนสมุดยุทธการที่วาดเล่นกันในวงเพื่อน แต่ interaction ต้องแม่นและอ่านง่าย

## Anti-references

ไม่เป็น dashboard องค์กร, จอเรดาร์ทหารจริงจัง, UI เกมมือถือที่เต็มไปด้วยแสงและ badge, หรือพื้นผิวตกแต่งที่ทำให้พิกัดและสถานะอ่านยาก

## Design Principles

- กระดานและคำสั่งต้องเป็นจุดสนใจหลัก
- ทุกกติกาสำคัญต้องมี feedback ก่อนผู้เล่นกดยืนยัน
- ความทำมือสร้างบุคลิก แต่ห้ามเปลี่ยนขนาด layout หรือทำให้ control คาดเดาไม่ได้
- ข้อมูลลับและข้อมูลสาธารณะต้องแยกชัดเจน
- ทุก action ที่ย้อนกลับได้ควรแก้ไขได้จากจุดที่ผู้เล่นลงมือ

## Accessibility & Inclusion

รองรับ keyboard focus, reduced motion, contrast สำหรับข้อความหลัก และ responsive widths ตั้งแต่ 320px โดยไม่ใช้สีเพียงอย่างเดียวในการบอกสถานะ
